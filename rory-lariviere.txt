Stretch Questions

    1. What is the difference between an inline element and a block element? [block elements always creat a new line after the closing tag whereas the inline ones do not.]
    2. What happens when an element is positioned absolutely? [it is positioned relative to the closest placed ancestor instead of the viewport. if one is not available it will use the document body and move along with the page while scrolling]
    3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? [flex-grow]
    4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default [display block: <p> and <h1> inline: <span> and <a> inline-block: <input>]
    5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? [its basically how every single html element has a box wrapped around it that you can manipulate to style the elements and webpage. it is made up of margin, border, padding, content. to 'fix' the box model you can use the box-sizing property to reintroduce the 'quirks' of pre css3 css, where an elements height and width is not affected by padding or borders. the box-sizing commands are content-box, padding-box, and border-box]
